Autonomous Line-Following Robot

Project Overview

Built an autonomous line-following robot using Arduino, infrared sensors, and a custom bracket during Academic Year 1. Responsible for programming control logic in C++. The robot successfully navigates marked lines using sensor feedback and algorithmic decision-making.

Key Features

  • Dual infrared sensors for line detection
  • Arduino-based microcontroller system
  • Custom 3D-printed bracket design
  • Proportional control algorithm
  • Real-time sensor processing
  • Efficient power management

Technical Implementation

The robot uses two infrared sensors mounted on the front chassis to detect the line. The Arduino processes sensor input and applies proportional control to adjust motor speeds, enabling smooth curve navigation. Custom C++ firmware optimizes sensor reading frequency and response time.

Technologies & Tools Used:

Arduino C++ Infrared Sensors Microcontroller Robotics
