Chainsaw-to-Model Engine Conversion

Project Overview

Converting a small combustion engine to electric start with a custom 3D-printed PLA crankcase. This project demonstrates hands-on mechanical expertise and advanced CAD design proficiency in component redesign and manufacturing.

Key Features

  • Custom 3D-printed PLA crankcase design
  • Electric start system integration
  • Engine tuning and optimization
  • Component compatibility testing
  • Iterative design improvements

Technical Challenges & Solutions

The main challenge was designing a crankcase that could handle the stresses of engine operation while being 3D printable. Through iterative design and material selection, a PLA-based solution was developed that provides adequate strength and thermal resistance.

Technologies & Tools Used:

SolidWorks 3D Printing CAD Design Engine Tuning Manufacturing
